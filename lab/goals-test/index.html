C: Spec Surgery

A product spec mixes goals and tasks. Cut every task — keep only goals. Can the product still be built?

Know if I need an umbrella today
GOAL

TASK

GOAL
Implement pull-to-refresh to update the radar map
TASK
Feel confident about what to wear
GOAL
Show temperature in both Fahrenheit and Celsius with a toggle
TASK
Add a hamburger menu with settings, about, and feedback options
TASK
Glance at my phone and know the weather in 2 seconds
GOAL
Click lines to cut them. Try to remove only tasks.

If you cut every task and the spec still makes sense, you had goals. If it collapses, you only had tasks.
